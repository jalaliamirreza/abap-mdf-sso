## نحوه استفاده

این فایل‌ها آماده آپلود به سایت تامین اجتماعی هستند:

1. به سایت تامین اجتماعی وارد شوید
2. بخش ارسال لیست بیمه را انتخاب کنید
3. هر دو فایل را آپلود کنید:
   - `dskkar00.dbf` (سرآیند)
   - `dskwor00.dbf` (کارگران)
4. فایل‌ها را ارسال کنید

## چگونه این فایل‌ها ایجاد شدند؟

```bash
# 1. آماده کردن فایل‌های CSV
# header.csv - اطلاعات کارگاه
# workers.csv - لیست کارگران

# 2. تبدیل به DBF
python tools/csv_to_dbf_complete.py \
    header.csv workers.csv \
    --workshop-id 9018282956 \
    --year 4 \
    --month 7 \
    --list-no 202407001 \
    --output-dir sample_output
```
