## [1.0.0] - 2024-01-15

### اضافه شده / Added

#### SAP ABAP Program
- برنامه ABAP `ZHCM_SSO_EXTRACT` برای استخراج داده از HCM
- پشتیبانی از تمامی Infotypes استاندارد HR
- محاسبه خودکار روزهای کارکرد
- استخراج اطلاعات payroll از نتایج
- خروجی JSON برای پردازش بیشتر
- پشتیبانی از Selection Screen با پارامترهای کامل
- امکان اجرای Batch Job

#### Python Utilities
- `generate_dbf.py`: تبدیل JSON به DBF
- `jalali_converter.py`: تبدیل تاریخ میلادی به شمسی
- اعتبارسنجی کد ملی با الگوریتم checksum
- اعتبارسنجی تاریخ شمسی
- پشتیبانی از کدگذاری Windows-1256

#### Configuration
- فایل تنظیمات SAP (`sap-config.template.json`)
- نگاشت فیلدها (`field_mappings.json`)
- تنظیمات Wage Types
- نگاشت انواع قرارداد

#### Documentation
- README جامع (دوزبانه فارسی/انگلیسی)
- راهنمای نصب (`INSTALLATION.md`)
- راهنمای استفاده (`USAGE.md`)
- مشخصات فرمت تامین اجتماعی (`SSO_FORMAT_SPEC.md`)
- CHANGELOG

#### Testing
- داده‌های نمونه (10 رکورد)
- تست‌های واحد (Unit Tests)
- تست‌های یکپارچگی (Integration Tests)
- اعتبارسنجی کد ملی
- اعتبارسنجی تاریخ

#### Project Structure
```
abap-mdf-sso/
├── src/
│   ├── abap/          # برنامه‌های ABAP
│   ├── utils/         # ابزارهای Python
│   └── transform/     # منطق تبدیل
├── config/            # تنظیمات
├── docs/              # مستندات
├── tests/             # تست‌ها
├── README.md
├── requirements.txt
└── CHANGELOG.md
```

## یادداشت‌های نسخه / Version Notes

### نسخه‌گذاری سمانتیک / Semantic Versioning

این پروژه از [Semantic Versioning](https://semver.org/) استفاده می‌کند:

- **MAJOR** (اصلی): تغییرات ناسازگار در API
- **MINOR** (فرعی): قابلیت‌های جدید با سازگاری به عقب
- **PATCH** (وصله): رفع باگ‌ها

### دسته‌بندی تغییرات / Change Categories

- **Added** (اضافه شده): قابلیت‌های جدید
- **Changed** (تغییر یافته): تغییرات در قابلیت‌های موجود
- **Deprecated** (منسوخ شده): قابلیت‌هایی که به زودی حذف می‌شوند
- **Removed** (حذف شده): قابلیت‌های حذف شده
- **Fixed** (رفع شده): رفع باگ‌ها
- **Security** (امنیتی): رفع آسیب‌پذیری‌های امنیتی
